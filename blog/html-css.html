DEFINITIONS

Classes and IDs provide similar but different functions that provide powerful funtions for the interaction between HTML and CSS.

Class
Assigning a class to an element provides you with the ability to apply specific CSS or JavaScript effects to the web page shown in general, a specific section ro multiple places. A class differs from an ID in that it can be repeated anywhere you choose by simply adding the class to the element.
